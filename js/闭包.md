# 闭包
> **闭包**：是指有权访问另一个函数作用域中的变量的函数;(摘自:**javascript高程**)


## 创建一个闭包
```js
function funA(a) {
  return function (b) {
    if(a>b) {
      return a - b;
    } else {
      return a + b;
    }
  }
}
const computed = funA(10);
const result = add(8);
```
**简单来说:funA的作用域中的变量a被内部的返回函数被调用了(涉及到了作用域链)而funA的返回函数被保存在了result变量中以至于funA中的变量a无法被浏览器的‘回收机制’正常回收(销毁)**
