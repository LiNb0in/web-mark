# 闭包
> **闭包**：是指有权访问另一个函数作用域中的变量的函数;(摘自:**javascript高程**)


## 创建一个闭包
```js
function funA(a) {
  return function (b) {
    if(a>b) {
      return a - b;
    } else {
      return a + b;
    }
  }
}
const computed = funA(10);

```
