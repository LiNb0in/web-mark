# apply、bind、call
> 它们都可以用来改变this的指向
## apply
  * 介绍
    * 语法
      > fun.apply(this, [args])。
    * 参数
      this: 可选，fun函数运行时使用的this，注意：**在非严格模式下当this为null或undefined时将会自动指向全局对象**
      args: 可选，一个数组或者类数组对象，其中的数组元素将作为单独的参数传给fun函数。
    * 返回值
      调用有指定this值和参数的函数的结果。
